# Book Sorter
Simple script that sorts a properly formatted .csv file containing information about books (or any .csv containing a column called 'title') by title alphabetically. It then prints the titles in order, then a count.

### Usage
`$ python3 book_sorter.py <optional/path/to/.csv>`
If no path is passed in, then book_sorter will look in the current directory for a file claled `book_list.csv`
